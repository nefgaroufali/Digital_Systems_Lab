PROJECT FILES

Χώρισα τα παραδοτέα σε φακέλους για να τα βρείτε πιο εύκολα. Μονάδες που αποτελούνταν από
επιμέρους modules και συμμετείχαν στην κατασκευή άλλων μερών, μπήκε στο καιινούριο μέρος
μόνο το τοπ module τους. Έτσι παραδείγματος χάριν στον τρανσμίτερ και στον ρισιβερ που περιέχουν
τον baud controller ο οποίος κατασκευάζεται και από άλλα modules, μπήκε μόνο το top module του
baud controller. Αντιστοιχα στο τελευταίο μέρος που γινεται η ένωση του transmiter και του
receiver μπηκαν μόνο τα top modules τους και όχι όλες οι μονάδες, για να είναι πιο εύκολο στην
διόρθωση. Επισης μεσα στον φακελο του καθε μερους περιεχονται και οι μοναδες ελεγχου για το
συγκεκριμενο μερος


PROJECT DATAFLOWS

Υπαρχουν επισης φακελοι με τα dataflows των μερων σε περιπτωση που δεν ειναι ορατα στο report.
Ο ενας φακελος περιεχει μορφη εικονας ενω ο αλλος μορφη που μπορει να ανοιχτει στο draw.io
Μορφη pdf δεν μπορουσε να απεικονισει καλα τα διαγραμματα.



Τελος εχει συμπεριληφθεί ενα βιντεο που δειχνει το πειραμα στην πλακετα για το προαιρετικο μερος.
