Τα αρχεία FourDigitLEDdriver, TritoMeros & TetartoMeros είναι τα topmodules για
το δεύτερο, τρίτο και τέταρτο μέρος αντιστοίχα. Τα αντίστοιχα testbench είναι
για τον έλεγχο των αντίστοιχων μερών. Τα υπόλοιπα modules είναι για τις υπόλοιπες
μονάδες οι οποίες εμφανίζονται με τα ίδια ονόματα στο report και στο dataflow.
Επιπλέον υπάρχουν μερικά testbench τα οποία είχα φτιάξει για να ελέγξω την
λειτουργία μερικών από αυτές τις επιμέρους μονάδες.
